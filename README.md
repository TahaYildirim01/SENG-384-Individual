# Person Management System with Docker

This project is a full-stack web application built with React, Express.js, PostgreSQL, Docker, and Docker Compose.

## Features
- Add new person
- View all people
- Update existing person
- Delete person
- Frontend validation
- Backend validation
- Unique email constraint
- Dockerized full-stack app

## Technologies
- React
- Node.js / Express
- PostgreSQL
- Docker
- Docker Compose

## Run the Project

```bash
docker compose up --build
